# Modelo de Gestión v2.0 — SaaS / B2B

Versión 2.0 del modelo de gestión empresarial. Identidad visual SaaS (slate + indigo). Orientado a empresas de Servicios Profesionales, Consultoría y SaaS.

## Módulos

1. Dashboard Ejecutivo
2. Ingresos
3. Costos Fijos
4. Costos Variables
5. Utilidad & Punto de Equilibrio
6. Sensibilización de Escenarios
7. Balance General
8. Cartera de Productos / Servicios + BCG
9. **Recursos Humanos** (nuevo: plantilla, costo, productividad, rotación)
10. Abastecimiento + N° Documento
11. Marketing Mix 7P
12. Compliance Legal

## Stack

Vite + React 18 + Tailwind CSS 3 + Recharts.

## Ejecutar local

```bash
npm install
npm run dev
```

## Deploy en Vercel

Conectar el repo de GitHub a Vercel; el `vercel.json` ya tiene la config lista.
